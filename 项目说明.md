# 课表管理系统

纯前端课表管理应用，iOS Liquid Glass 风格设计。

## 快速启动

```bash
cd D:\Cluadecode\课表项目
python -m http.server 3000
# 浏览器访问 http://localhost:3000
```

## 课表数据

- 来源：学校官网粘贴的 HTML 表格 → Node 脚本解析 → 25 条课程嵌入 JS
- 周次范围：1-17 周，每周课程不同
- 时间段：5 个时段（第一二节 ~ 第十十一节，10:00-21:30）
- 涵盖周一至周日

## 功能

| 功能 | 操作 |
|------|------|
| 周视图课表 | 7列×5行网格，按周筛选课程 |
| 今日高亮 | 当天列紫色标记 + "今天"标识 |
| 周次切换 | `←` `→` 键盘、下拉菜单、❤️按钮 |
| 课程详情 | 点击课程卡片弹窗（教师/教室/时间/周次） |
| 暗色/亮色主题 | 右上角按钮切换，localStorage 记忆 |

## 设计

- **风格**：iOS Liquid Glass（液态玻璃）
- **效果**：全站 `backdrop-filter` 毛玻璃、胶囊按钮、弹性缩放 hover、Canvas 粒子动画
- **壁纸**：`body::before` 伪元素加载背景图
- **响应式**：移动端自适应

## 技术栈

- 纯前端：HTML + CSS + JavaScript
- 数据存储：嵌入式 JSON（`COURSES_DATA` 常量）
- 主题持久化：localStorage
